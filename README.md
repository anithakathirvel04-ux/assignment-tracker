# Assignment Submission Tracker

A React.js dashboard for college faculty to track assignment submissions.

## Features
- Sign up / Sign in (stored locally per-browser)
- Add assignments (title, subject, due date)
- Mark status: Submitted / Pending / Late
- Filter by subject
- Dashboard summary with counts
- Beautiful, animated, responsive UI

## Run locally
```bash
npm install
npm run dev
```
Then open http://localhost:5173

## Build
```bash
npm run build
```
